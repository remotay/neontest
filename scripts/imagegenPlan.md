# Image Generation Plan

All project visual assets are generated with Codex's built-in `image_gen` tool and finalized into `public/assets/images`.

Global style prefix used for every generated background:

```text
high quality surreal pixel art horror, 16-bit/32-bit hybrid pixel aesthetic, CRT scanline texture, abstract religious cyberpunk, eldritch atmosphere, neon magenta cyan acid yellow on deep black, gritty dithering, corrupted stained glass, occult machine details, oppressive composition, no clean cartoon style, no anime, no readable real-world text, browser game background, 1280x720
```
